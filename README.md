# FRET
an [ImageJ](http://imagej.net/Welcome)/[Fiji](https://fiji.sc) python script  used to analyze spectral confocal microscopy images and to quantify [FRET](https://en.wikipedia.org/wiki/Förster_resonance_energy_transfer) (Förster Resonance Energy Transfer) signals. The script can process either a single multichannel spectral LSM file or two separate donor and acceptor image files and provides a flexible workflow for preprocessing, background subtraction, photobleaching correction, thresholding, and FRET metric calculation. A step-by-step visual guidance of the script is shown in the Figure 4.at the nuclear envelope or in the cytoplasm for FRET standard constructs.
